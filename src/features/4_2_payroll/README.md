# Payroll Module

This module handles payroll expense functionality for the expenses system.

## Components

- **PayrollExpensePage** - Main page for payroll expense management
- **PayrollExpenseFilters** - Filters for payroll expenses
- **PayrollExpenseMetricsCards** - Key metrics display for payroll expenses
- **PayrollExpenseTable** - Table displaying payroll expenses
- **PayrollExpenseOverview** - Overview of payroll expenses

## Usage

This module is used in the `/expenses/payroll` route and provides comprehensive payroll expense management functionality.

## Dependencies

- Re-exports hooks from the main expenses dashboard module
- Uses shared UI components from the design system
