# Sales Activities Module

This module handles sales activities functionality for the operations system.

## Components

- **SalesActivitiesTable** - Table displaying sales activities
- **SalesActivitiesPageContent** - Main page content for sales activities
- **SalesActivitiesOverview** - Overview of sales activities
- **SalesActivitiesMetricsCards** - Key metrics display for sales activities
- **SalesActivitiesFilters** - Filters for sales activities
- **SalesActivitiesActionsDropdown** - Actions dropdown for sales activities
- **SalesActivityDialog** - Dialog for creating/editing sales activities
- **SalesActivityForm** - Form for sales activity data
- **SalesActivityItemsManager** - Manager for sales activity items
- **AddItemDialog** - Dialog for adding items to sales activities
- **ItemsTable** - Table for displaying items

## Usage

This module is used in the `/operations/sales/activities` route and provides comprehensive sales activities management functionality.

## Dependencies

- Uses shared UI components from the design system
- Integrates with sales activities hooks and data management
