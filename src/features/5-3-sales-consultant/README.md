# Sales Consultant Module

This module handles sales consultant management functionality for the consultant system.

## Components

### Main Consultant View
- **ConsultantsTableViewContent** - Main table view for sales consultants

### Management Dialogs
- **CategoriesManagementDialog** - Dialog for managing lead categories
- **ServicesManagementDialog** - Dialog for managing services
- **SourceManagementDialog** - Dialog for managing lead sources
- **StatusManagement** - Component for managing consultant statuses

### Supporting Components
- **ClientProfilePopup** - Popup for displaying client profiles
- **DateRangeFilter** - Filter component for date ranges

## Usage

This module is used in the `/operations/consultant/consultants` route and provides comprehensive sales consultant management functionality including:

- Consultant performance tracking
- Lead source management
- Service category management
- Status tracking and updates
- Client profile management
- Advanced filtering capabilities

## Dependencies

- Uses shared UI components from the design system
- Integrates with consultant data management
- Supports advanced filtering and search functionality
- Includes management dialogs for data configuration
