# Customer Service Dashboard Module

This module handles customer service dashboard functionality for the operations system.

## Components

- **CustomerServiceDashboard** - Main dashboard page for customer service
- **CustomerServiceTicketsPage** - Tickets management page
- **CustomerServiceTicketsTable** - Table displaying customer service tickets
- **CustomerServiceTicketsFilters** - Filters for tickets
- **CustomerServiceMetricsCards** - Key metrics display for customer service
- **CustomerServiceInsights** - Insights and analytics
- **DashboardNewTicketButton** - Button for creating new tickets
- **CustomerServiceTicketActionsDropdown** - Actions dropdown for tickets
- **CreateTicketDialog** - Dialog for creating new tickets
- **EditTicketDialog** - Dialog for editing tickets

## Usage

This module is used in the `/operations/customer-service/dashboard` route and provides comprehensive customer service management functionality.

## Dependencies

- Uses shared UI components from the design system
- Integrates with customer service hooks and data management
