# Customer Service Tickets Module

This module handles customer service tickets functionality for the operations system.

## Components

- **CustomerServiceTicketsPage** - Main tickets management page
- **CustomerServiceTicketsTable** - Table displaying customer service tickets
- **CustomerServiceTicketsFilters** - Filters for tickets
- **CustomerServiceTicketActionsDropdown** - Actions dropdown for tickets
- **CreateTicketDialog** - Dialog for creating new tickets
- **EditTicketDialog** - Dialog for editing tickets

## Usage

This module is used in the `/operations/customer-service/tickets` route and provides comprehensive ticket management functionality.

## Dependencies

- Uses shared UI components from the design system
- Integrates with customer service hooks and data management
