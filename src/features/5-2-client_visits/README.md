# Client Visits Module

This module handles client visits functionality for the operations system.

## Components

- **ClientVisitsPage** - Main page for client visits management
- **ClientVisitsTable** - Table displaying client visits
- **ClientVisitsFilters** - Filters for client visits
- **ClientVisitsMetricsCards** - Key metrics display for client visits
- **RecentClientVisitsOverview** - Overview of recent client visits

## Usage

This module is used in the `/operations/sales/client-visits` route and provides comprehensive client visits management functionality.

## Dependencies

- Uses shared UI components from the design system
- Integrates with client visits hooks and data management
