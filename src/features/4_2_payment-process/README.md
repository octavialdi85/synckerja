# Payment Process Module

This module handles payment processing functionality for the expenses system.

## Components

- **PaymentProcessPage** - Main page for payment processing
- **PaymentFilters** - Filters for payment requests
- **PaymentMetricsCards** - Key metrics display for payments
- **PaymentRequestsTable** - Table displaying payment requests
- **PaymentRequestDetailDialog** - Dialog for viewing payment request details
- **RecentPaymentsOverview** - Overview of recent payments

## Usage

This module is used in the `/expenses/payment-process` route and provides comprehensive payment processing functionality.

## Dependencies

- Re-exports hooks from the main expenses dashboard module
- Uses shared UI components from the design system
