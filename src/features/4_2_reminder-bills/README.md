# Reminder Bills Module

This module handles reminder bills functionality for the expenses system.

## Components

- **ReminderBillsPage** - Main page for reminder bills management
- **ReminderBillsFilters** - Filters for reminder bills
- **ReminderBillsMetricsCards** - Key metrics display for reminder bills
- **ReminderBillsTable** - Table displaying reminder bills
- **ReminderBillsOverview** - Overview of reminder bills

## Usage

This module is used in the `/expenses/reminder-bills` route and provides comprehensive reminder bills management functionality.

## Dependencies

- Re-exports hooks from the main expenses dashboard module
- Uses shared UI components from the design system
